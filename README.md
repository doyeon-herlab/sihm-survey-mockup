# SIHM 근골격계 설문조사 목록 — 재기획 목업

근골격계 설문조사 목록 화면의 재기획안을 개발 전달용으로 시각화한 정적 목업입니다. 실제 배포본이 아니라 의도 전달용 더미 데이터 화면입니다.

👉 미리보기: `index.html`을 브라우저로 열거나 GitHub Pages로 확인하세요.

## 재기획 핵심

- **응답일시 컬럼 신설** — 나이와 판정 사이에 응답일시(서버 수신 시각)를 표시.
- **응답 누적 방식** — 동일 이름·나이여도 제출할 때마다 별도 행으로 쌓이며, 응답일시 오름차순(최신이 맨 아래)으로 정렬. 각 행 = 1건의 응답(`response_id`).
- **무증상 표기** — 판정이 무증상이면 배지 없이 `-`로 표시.
- **응답률 차트** — `응답 건수 / 전체 응답 대상자 수`. `100%` 이하는 초록(`#48AA66`), `100%` 초과는 파랑(`#0063F7`)으로 표시(예: 7/4 → 175% 파랑).

## 디자인 토큰

`SIHM Style Guide`를 따름 — Pretendard 폰트, MDI 아이콘, Primary `#48AA66`, Grey scale, Table / Button / Table Label / 부서 트리 규칙.

- 더미 데이터·렌더링 로직은 `index.html` 하단 `<script>`에 인라인.
